# 🍳 SVGE Recipes – Recipe Finder & Meal Planner

**SVGE Recipes** is a full‑featured web application that lets users search for recipes, filter by category, save favorites, plan weekly meals, and view nutritional information. It’s built with vanilla HTML, CSS, and JavaScript, and integrates two powerful APIs:

- [TheMealDB](https://www.themealdb.com) – for recipe discovery, search, and meal data.
- [Edamam Nutrition API](https://developer.edamam.com/edamam-docs-recipe-api) – for real‑time nutritional analysis (optional but recommended).

The app is fully responsive and works on both desktop and mobile devices. All user data (favorites and meal plans) is stored locally in your browser.

---

## ✨ Features

- 🔍 **Search recipes** by name (e.g., “chicken”, “pasta”, “salad”)
- 🏷️ **Filter by category** (Beef, Chicken, Dessert, Seafood, etc.)
- ❤️ **Save favorites** – persists in your browser’s local storage
- 📅 **Weekly meal planner** – add/remove meals for each day
- 🥗 **Nutritional information** – calories, protein, fat, carbs (powered by Edamam)
- 🖱️ **Click any recipe** to see its full ingredient list and nutrition (if API keys configured)
- 📱 **Responsive design** – works on phones, tablets, and desktops
